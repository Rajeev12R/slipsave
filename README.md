# SlipSave

**Turn your spare change into real wealth — automatically.**

---

## Slogan

**"Wealth from what you won’t miss."**  
SlipSave helps you invest a tiny fraction (like 0.1%) of every UPI transaction — the part you'd never notice — into a secure, intelligent investment wallet.

---

## Why I'm Building It

Most people want to save or invest, but don’t — not because they’re careless, but because the habit is hard to build and maintain.  
Yet, we spend money every day — and often forget about the ₹1–₹3 change amounts.

**SlipSave removes the friction** by automating investments from a fraction of each payment — amounts that are emotionally negligible but financially powerful over time.

Instead of asking people to save, we just work with what they’re already spending.

---

## Real-World Impact

### Meet Aryan — a 22-year-old engineering student.

Aryan uses UPI 4–5 times a day. A cab for ₹73.80, chai for ₹18.50, and Zomato order for ₹189.25.  
He enables SlipSave to auto-invest **0.1% of every spend**.

In a week, he has unknowingly invested:
₹0.07 + ₹0.01 + ₹0.19 + ₹0.08 + ₹0.21 + ₹0.04 = ₹0.60/day avg = ₹4.20/week

> In 1 year, this "invisible" ₹0.60/day becomes ~₹219.  
> With passive interest (7%), it grows even more.  
> In 5 years — **he builds ₹1,300+ from literal spare change**.

## Key Features

| Feature | Description |
|--------|-------------|
| **Customizable % Investing** | User chooses how much of each transaction to invest — e.g. 0.1%, 0.25%, etc. |
| **Transaction Tracker** | Reads transactions via SMS or mock APIs to trigger investment logic. |
| **Forgettable Investing** | Invests only the amount that feels insignificant, e.g. ₹0.07 or ₹0.28. |
| **Smart Wallet** | Shows real-time portfolio, passive savings total, and visual projections. |
| **Visual Growth Tracker** | Graphs and charts help users see long-term growth of their micro-investments. |
| **Goal-Based Investing** | Optional: set investment goals (trip, emergency fund, etc.) |
| **Gamified Badges (Optional)** | Reward users for milestones — “₹100 Club”, “Invisible Investor”, etc. |
| **Privacy-First Design** | No sensitive banking data stored; works on-device or with user consent. |
